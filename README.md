# 📊 Трекер расходов

Таблица для ведения еженедельных расходов.

## 📋 Параметры

- **Недельный лимит:** 7000 рублей
- **Колонки:** Дата, Категория, Сумма

## ��� Как использовать

1. Откройте файл `expenses.csv`
2. Добавляйте новые строки с вашими расходами
3. Заполняйте: дату, категорию, сумму
4. Отслеживайте остаток бюджета

## 📂 Формат файла

```
Дата,Категория,Сумма
2026-04-28,Еда,350
2026-04-28,Транспорт,200
```

## 💡 Рекомендуемые категории

- 🍔 Еда
- 🚗 Транспорт
- 🏠 Жилище
- 🎮 Развлечения
- 🛍️ Покупки
- 💊 Здоровье
- 📚 Образование
- Другое

**Удачи с ведением бюджета!** 💰
